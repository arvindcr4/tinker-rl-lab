          <pre class="python"><span class="hljs-string">'''
build profile graph for the given instance

running:
  $ get_gprof &lt;args&gt; &lt;instance&gt;

executes:
  gprof2dot -f pstats &lt;args&gt; &lt;type&gt;.prof | dot -Tpng -o &lt;type&gt;.call.png

where:
  &lt;args&gt; are arguments for gprof2dot, such as "-n 5 -e 5"
  &lt;instance&gt; is code to create the instance to profile
  &lt;type&gt; is the class of the instance (i.e. type(instance))

For example:
  $ get_gprof -n 5 -e 1 "import numpy; numpy.array([1,2])"

will create 'ndarray.call.png' with the profile graph for numpy.array([1,2]),
where '-n 5' eliminates nodes below 5% threshold, similarly '-e 1' eliminates
edges below 1% threshold
'''</span>
