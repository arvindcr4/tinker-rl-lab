    <div class="docs markdown"><div class="pilwrap" id="tinker-rl-lab">
  <h1>
    <a href="#tinker-rl-lab" name="tinker-rl-lab" class="pilcrow"></a>
Tinker RL Lab
  </h1>
</div>
<p>A consolidated research repository for Reinforcement Learning experiments with Large Language Models, integrating the Tinker platform, Atropos environments, and RL Gym implementations.</p>
<div class="pilwrap" id="overview">
  <h2>
    <a href="#overview" name="overview" class="pilcrow"></a>
Overview
  </h2>
</div>
<p>This repository consolidates multiple research projects focused on:</p>
<ul>
<li><strong>Tinker Platform Experiments</strong>: RL training, SFT, preference learning, and distillation</li>
<li><strong>Atropos Integration</strong>: Connecting Atropos environments with Tinker API</li>
<li><strong>RL Gym</strong>: Google Drive and browser automation tasks for RL research</li>
</ul>
<div class="pilwrap" id="repository-structure">
  <h2>
    <a href="#repository-structure" name="repository-structure" class="pilcrow"></a>
Repository Structure
  </h2>
</div>
<pre><code>tinker-rl-lab/
├── grpo-results/         # GRPO RL experiment results and metrics
├── agentic-rl-finetuning/ # Agentic RL fine-tuning research
├── capstone-literature-survey/ # Literature Survey: RL for LLMs (GRPO Scaling)
├── experiments/           # Tinker RL Cookbook experiments
│   ├── notebooks/        # Jupyter notebooks for each experiment
│   ├── implementations/  # RL implementations (PPO, DPO, GRPO, etc.)
│   ├── results/          # Training metrics
│   └── logs/             # Training output logs
│
├── atropos/              # Tinker-Atropos integration
│   ├── tinker_atropos/   # Core package
│   │   ├── environments/ # GSM8K, Math, LogP steering
│   │   ├── trainer.py    # Tinker trainer
│   │   └── config.py     # Configuration management
│   ├── configs/          # YAML configurations
│   └── notebooks/        # Analysis notebooks
│
├── rl-gym/               # RL Gym (Google Drive &amp; Browser tasks)
│   ├── gcloud_drive/     # Google Drive integration
│   └── docs/             # Task documentation
│
└── rl-master/            # RL Master (Task execution)
    ├── client_for_your_service.py
    └── gcloud_drive/
</code></pre>

<p><strong>Key Results:</strong></p>
<ul>
<li>Arithmetic: 69.5% → 100% accuracy in ~20 steps</li>
<li>Preference learning effectively shapes response style</li>
<li>Distillation transfers knowledge efficiently</li>
</ul>
<div class="pilwrap" id="2.atropos-integration">
  <h3>
    <a href="#2.atropos-integration" name="2.atropos-integration" class="pilcrow"></a>
2. Atropos Integration
  </h3>
</div>
<p>Integration layer connecting <a href="https://github.com/NousResearch/atropos">Atropos</a> with the Tinker API.</p>
<p>Features:</p>
<ul>
<li>Use any Atropos environment with Tinker training</li>
<li>Built-in GSM8K and Math environments</li>
<li>LoRA-based fine-tuning with configurable parameters</li>
<li>Checkpoint management and weight downloading</li>
</ul>
<div class="pilwrap" id="3.rl-gym">
  <h3>
    <a href="#3.rl-gym" name="3.rl-gym" class="pilcrow"></a>
3. RL Gym
  </h3>
</div>
<p>Google Drive and browser automation tasks for RL agent training.</p>
<p>Features:</p>
<ul>
<li>200+ tasks for browser automation</li>
<li>Google Drive API integration</li>
<li>MCP tool integration</li>
<li>Task generation and verification</li>
</ul>

<div class="pilwrap" id="prerequisites">

Prerequisites
  </h3>
</div>
<pre><code class="bash"><span class="hljs-comment"># Create virtual environment</span>
python3 -m venv tinker-env
<span class="hljs-built_in">source</span> tinker-env/bin/activate

<span class="hljs-comment"># Install dependencies</span>
pip install tinker tinker-cookbook atropos
</code></pre>
<div class="pilwrap" id="running-tinker-experiments">
  <h3>
    <a href="#running-tinker-experiments" name="running-tinker-experiments" class="pilcrow"></a>
Running Tinker Experiments
  </h3>
</div>
<pre><code class="bash"><span class="hljs-built_in">export</span> TINKER_API_KEY=<span class="hljs-string">"your-key-here"</span>

<span class="hljs-comment"># Math RL</span>
python -m tinker_cookbook.recipes.math_rl.train \
    model_name=<span class="hljs-string">"meta-llama/Llama-3.2-1B"</span> \
    env=arithmetic

<span class="hljs-comment"># Chat SFT</span>
python -m tinker_cookbook.recipes.chat_sl.train \
    model_name=<span class="hljs-string">"meta-llama/Llama-3.2-1B"</span>
</code></pre>
<div class="pilwrap" id="running-atropos-+-tinker">
  <h3>
    <a href="#running-atropos-+-tinker" name="running-atropos-+-tinker" class="pilcrow"></a>
Running Atropos + Tinker
  </h3>
</div>
<pre><code class="bash"><span class="hljs-comment"># Terminal 1: Start Atropos API</span>
run-api

<span class="hljs-comment"># Terminal 2: Start training</span>
<span class="hljs-built_in">export</span> TINKER_API_KEY=<span class="hljs-string">"your-key"</span>
python atropos/launch_training.py --config atropos/configs/default.yaml

<span class="hljs-comment"># Terminal 3: Start environment</span>
python atropos/tinker_atropos/environments/gsm8k_tinker.py serve \
    --config atropos/configs/default.yaml
</code></pre>
