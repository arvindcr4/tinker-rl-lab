<p>Start a local inference server for a base model or saved checkpoint, then run
the deterministic reasoning benchmark suite against it.</p>
<p>Examples:
./run_claim_support_evals.sh --model Qwen/Qwen3-8B
./run_claim_support_evals.sh --model Qwen/Qwen3-8B --weights tinker://.../step_50
./run_claim_support_evals.sh --config configs/gsm8k_qwen_8b.yaml --weights tinker://... --no-prefix</p>
