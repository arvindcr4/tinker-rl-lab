<p>Run a single GSM8K GRPO experiment end-to-end
Usage: ./run_experiment.sh configs/gsm8k_llama_3b.yaml</p>
<p>This script starts all 3 required processes:</p>
<ol>
<li>Atropos API server (run-api)</li>
<li>Tinker trainer (launch_training.py)</li>
<li>GSM8K environment (gsm8k_tinker.py serve)</li>
</ol>
<p>And waits for the trainer to finish, then cleans up.</p>

        </td>
        <td class="code highlight">
          <pre class="sh">
