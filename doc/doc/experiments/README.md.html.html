  <h1>

Tinker RL Cookbook Experiments
  </h1>
</div>
<p>PES LLM Research Project - Reinforcement Learning for Language Models</p>
<div class="pilwrap" id="overview">
  <h2>
    <a href="#overview" name="overview" class="pilcrow"></a>
Overview
  </h2>
</div>
<p>This repository contains experiments with the <a href="https://thinkingmachines.ai/tinker">Tinker</a> platform for training LLMs using reinforcement learning, supervised fine-tuning, and knowledge distillation.</p>

<div class="pilwrap" id="setup">
  <h2>
    <a href="#setup" name="setup" class="pilcrow"></a>
Setup
  </h2>
</div>
<ol>
<li>Create a virtual environment:</li>
</ol>
<pre><code class="bash">python3 -m venv tinker-env
<span class="hljs-built_in">source</span> tinker-env/bin/activate
</code></pre>
<ol start="2">
<li>Install dependencies:</li>
</ol>
<pre><code class="bash">pip install tinker tinker-cookbook
</code></pre>
<ol start="3">
<li>Set your API key:</li>
</ol>
<pre><code class="bash"><span class="hljs-built_in">export</span> TINKER_API_KEY=<span class="hljs-string">"your-key-here"</span>
</code></pre>

<div class="pilwrap" id="math-rl-arithmetic">
  <h3>
    <a href="#math-rl-arithmetic" name="math-rl-arithmetic" class="pilcrow"></a>
Math RL (Arithmetic)
  </h3>
</div>
<pre><code class="bash">python -m tinker_cookbook.recipes.math_rl.train \
    model_name=<span class="hljs-string">"meta-llama/Llama-3.2-1B"</span> \
    env=arithmetic \
    group_size=4 \
    groups_per_batch=100 \
    learning_rate=1e-4
</code></pre>
<div class="pilwrap" id="chat-supervised-learning">
  <h3>
    <a href="#chat-supervised-learning" name="chat-supervised-learning" class="pilcrow"></a>
Chat Supervised Learning
  </h3>
</div>
<pre><code class="bash">python -m tinker_cookbook.recipes.chat_sl.train \
    model_name=<span class="hljs-string">"meta-llama/Llama-3.2-1B"</span> \
    dataset=no_robots \
    learning_rate=5e-4 \
    batch_size=32
</code></pre>

Learning Curves
  </h3>
