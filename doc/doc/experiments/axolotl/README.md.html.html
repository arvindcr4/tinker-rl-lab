  <h1>
    <a href="#axolotl-experiment-configs" name="axolotl-experiment-configs" class="pilcrow"></a>
Axolotl Experiment Configs
  </h1>
</div>
<p>Matched baselines for the GRPO scaling study, runnable on HuggingFace Spaces,
RunPod, or any GPU server. These address reviewer criticism #8 (&quot;no matched
baseline showing GRPO is the reason&quot;) and #15 (&quot;GRPO is obsolete&quot;).</p>

Quick Start

</div>
<div class="pilwrap" id="install-axolotl">
  <h3>
    <a href="#install-axolotl" name="install-axolotl" class="pilcrow"></a>
Install Axolotl
  </h3>
</div>
<pre><code class="bash">pip install axolotl[vllm]
</code></pre>
<div class="pilwrap" id="sft-baseline-single-gpu">
  <h3>
    <a href="#sft-baseline-single-gpu" name="sft-baseline-single-gpu" class="pilcrow"></a>
SFT Baseline (single GPU)
  </h3>
</div>
<pre><code class="bash"><span class="hljs-comment"># QLoRA on A10G / T4 (24 GB)</span>
axolotl train experiments/axolotl/sft_gsm8k_qwen8b.yaml

<span class="hljs-comment"># Full LoRA on A100 (80 GB)</span>
axolotl train experiments/axolotl/sft_gsm8k_qwen8b_full_lora.yaml
</code></pre>
<div class="pilwrap" id="grpo-2-gpus">
  <h3>
    <a href="#grpo-2-gpus" name="grpo-2-gpus" class="pilcrow"></a>
GRPO (2 GPUs)
  </h3>
</div>
<pre><code class="bash"><span class="hljs-comment"># Terminal 1: vLLM server on GPU 1</span>
CUDA_VISIBLE_DEVICES=1 axolotl vllm-serve experiments/axolotl/grpo_gsm8k_qwen8b.yaml

<span class="hljs-comment"># Terminal 2: Training on GPU 0</span>
CUDA_VISIBLE_DEVICES=0 axolotl train experiments/axolotl/grpo_gsm8k_qwen8b.yaml
</code></pre>
<div class="pilwrap" id="dr.grpo-variant-2-gpus">

Dr. GRPO variant (2 GPUs)
  </h3>

<pre><code class="bash">CUDA_VISIBLE_DEVICES=1 axolotl vllm-serve experiments/axolotl/grpo_gsm8k_qwen8b_dr_grpo.yaml
CUDA_VISIBLE_DEVICES=0 axolotl train experiments/axolotl/grpo_gsm8k_qwen8b_dr_grpo.yaml
</code></pre>
<div class="pilwrap" id="running-on-huggingface-spaces">
  <h2>
    <a href="#running-on-huggingface-spaces" name="running-on-huggingface-spaces" class="pilcrow"></a>
Running on HuggingFace Spaces
  </h2>
</div>
<ol>
<li>Create a Space with <strong>A100 GPU</strong> ($2.50/hr) or <strong>2x A10G</strong> ($3.00/hr)</li>
<li>Install: <code>pip install axolotl[vllm]</code></li>
<li>Clone this repo and run the commands above</li>
</ol>
<div class="pilwrap" id="running-on-runpod-lambda">
  <h2>
    <a href="#running-on-runpod-lambda" name="running-on-runpod-lambda" class="pilcrow"></a>
Running on RunPod / Lambda
  </h2>
</div>
<ol>
<li>Rent an A100 80GB ($1.39-1.48/hr) or 2x A10G</li>
<li>Install: <code>pip install axolotl[vllm]</code></li>
<li>Clone and run</li>
</ol>
