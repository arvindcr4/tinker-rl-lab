<div class="pilwrap" id="important-notes">
  <h3>
    <a href="#important-notes" name="important-notes" class="pilcrow"></a>
Important Notes
  </h3>
</div>
<ol>
<li><strong>GRPO is best for LLM math</strong> - Native verifiable reward support</li>
<li><strong>DPO degrades over iterations</strong> - Limit to 1-2 rounds</li>
<li><strong>KL early stopping</strong> - Use <code>target_kl=0.01</code> in PPO</li>
<li><strong>SB3/CleanRL are for RL agents, not LLMs</strong> - Shown for pattern reference</li>
</ol>
