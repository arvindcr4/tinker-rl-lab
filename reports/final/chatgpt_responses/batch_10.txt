Q46. Practitioner Decision Flowchart (model size, task, data → GRPO config)

Decision axes & mapping:

Model Size

<3B → conservative policy learning; lower entropy bonus; small replay λ.

3–10B → balanced exploration/exploitation; moderate entropy & replay mixing.

>10B → more aggressive policy updates; higher entropy bonus for diverse behaviors.

Task Type

Classification/Seq2Seq → higher advantage normalization; smaller clip range.

Open generation/Dialogue → larger clip range; stronger entropy regularization.

Action sequences (tool calls) → longer horizons → truncated PG with larger batch lengths.

Data Quality & Size

High-quality human data → lower KL penalty; smaller trust region.

Noisy/task‑synthesized → higher KL penalty; strong trust region constraints.

Sparse reward signals → larger batch & longer training schedule.

Recommended config template (actionable):

if model_size < 3B:
    entropy_coef = 0.01
    kl_coef = 0.1
    batch_size = 64
elif 3B <= model_size <= 10B:
    entropy_coef = 0.05
    kl_coef = 0.05
    batch_size = 128
else:
    entropy_coef = 0.1
    kl_coef = 0.01
    batch_size = 256

if task_type == “open_gen”:
    clip_range = 0.3
else:
    clip_range = 0.2

if data_quality == “noisy”:
    kl_coef *= 2.0

Result: choice of entropy_coeff, kl_coeff, clip_range, batch_size tuned by these axes.

Q50. Quantifying Sub‑3B GRPO Failure (as Negative Result)

Metric definitions for reporting:

Success criterion: e.g., BLEU/ROUGE/Reward > baseline + δ

Confidence: 95% CI on metric via bootstrap or multiple seeds

Procedure:

Run ≥5 independent runs with distinct seeds.

Compute mean ± 95% CI of target metric.

Compare with baseline (e.g., static fine‑tune) with paired tests.

Quantify failure:

GRPO (mean) = μ
Baseline = b
Δ = μ − b
95% CI (low, high)
If (CI_high < b + δ) → statistically below engineering threshold
Report p‑value (e.g., paired t‑test)

Example statement:

“For models <3B, GRPO yielded mean reward 0.72 (95% CI [0.68, 0.76]) vs baseline 0.81. CI does not overlap baseline + δ=0.05, indicating failure at 95% confidence.”

Include effect sizes, CI, p‑values to make it citable as an engineering guideline.
