# Tinker RL Lab Submission Bundle

This zip contains the current code, paper PDFs, capstone report PDF, and curated supporting data generated from the working tree.

## Contents

- `paper.pdf`: current main paper.
- `paper_anon.pdf`: anonymized paper build.
- `grpo_agentic_llm_paper.pdf`: compact GRPO/agentic-LLM paper.
- `grpo_agentic_llm_paper_anonymous.pdf`: anonymized compact paper.
- `presentation.pptx`: sanitized final capstone presentation deck.
- `report.pdf`: capstone final report.
- `ethics_statement.pdf`: standalone ethics statement.
- `code.tar.gz`: source code and document sources, excluding generated archives, private caches, checkpoints, W&B state, and secrets.
- `supporting_data.tar.gz`: curated result JSON/CSV/MD artifacts, BrowserGym smoke outputs, result ledger, and a small private-audit manifest snapshot. The large local W&B/HF/Tinker mirror is not included.
- `checksums.sha256`: checksums for bundle members.
- `MANIFEST.md`: human-readable manifest.

Verify after unzip with `sha256sum -c checksums.sha256`.
