# Tinker RL Lab Submission Bundle

This zip contains the compact anonymous NeurIPS-style paper and anonymized code generated from the working tree.

## Contents

- `paper_anon.pdf`: compact anonymous diagnostic-audit paper built with the NeurIPS 2026 style.
- `code.tar.gz`: anonymized source code and document sources, excluding generated archives, private caches, checkpoints, W&B state, and secrets.
- `checksums.sha256`: checksums for bundle members.
- `MANIFEST.md`: human-readable manifest.

Verify after unzip with `sha256sum -c checksums.sha256`.
