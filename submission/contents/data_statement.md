# Data Statement

The bundle redistributes project-generated result artifacts only. External benchmark datasets such as GSM8K, MATH, HumanEval, MBPP, GSM-1k, BrowserGym/MiniWoB, and WebArena are referenced by scripts and metadata but are not redistributed here. The supporting-data archive contains run summaries, result tables, smoke-test outputs, and local audit manifests needed to inspect the reported claims.

Large binary model weights, local Hugging Face caches, private W&B history mirrors, and Tinker checkpoints are intentionally excluded from the zip. They are represented by manifests and result ledgers.
